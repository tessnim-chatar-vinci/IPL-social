## Project Description
IPL Social is a project developed using Node.js, Express and Typescript.
The goal of this project is to set up a simple DevOps environment with a Continuous Integration (CI) pipeline using GitHub Actions.

The project includes the implement of an email validation function written in TypeScript. The devlopment of this feature follows the Test Driven Development approach, meaning that unit tests are written before the implementation of the validation logic.No deployment to a server is included, as required by the assignement.
